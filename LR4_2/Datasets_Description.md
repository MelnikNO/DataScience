# Рекомендованные датасеты для лабораторной работы по временным рядам

## Введение

Этот документ содержит подборку датасетов временных рядов, подходящих для обучения студентов полноценному исследованию и прогнозированию. Датасеты подобраны с учетом различных уровней сложности и охватывают разные области применения.

---

## 1. Airline Passengers (Пассажиры авиалиний)

**Описание:** Классический датасет с количеством пассажиров авиалиний по месяцам с 1949 по 1960 год.

**Характеристики:**
- **Период:** 1949-1960 (12 лет)
- **Частота:** Месячная
- **Количество наблюдений:** 144
- **Тип:** Одномерный временной ряд
- **Особенности:** Явный тренд и сезонность

**Загрузка:**
```python
import seaborn as sns
df = sns.load_dataset('flights')
```

**Применение:** Идеален для начинающих. Демонстрирует базовые концепции: тренд, сезонность, стационарность.

---

## 2. Air Quality UCI

**Описание:** Данные о качестве воздуха с датчиков в итальянском городе. Содержит концентрации различных газов и метеорологические параметры.

**Характеристики:**
- **Период:** Март 2004 - Апрель 2005
- **Частота:** Почасовая
- **Количество наблюдений:** ~9000
- **Тип:** Многомерный временной ряд
- **Переменные:** CO, NO2, NOx, O3, температура, влажность и др.

**Загрузка:**
```python
url = 'https://archive.ics.uci.edu/ml/machine-learning-databases/00360/AirQualityUCI.zip'
# Скачать и распаковать
df = pd.read_csv('AirQualityUCI.csv', sep=';', decimal=',')
```

**Применение:** Многомерный анализ, работа с экзогенными переменными, обработка пропусков.

---

## 3. Electricity Consumption (Потребление электроэнергии)

**Описание:** Данные о потреблении электроэнергии различными потребителями.

**Характеристики:**
- **Период:** Зависит от источника (обычно несколько лет)
- **Частота:** Часовая/дневная
- **Тип:** Одномерный или многомерный
- **Особенности:** Сильная сезонность (суточная, недельная, годовая)

**Загрузка:**
```python
# Пример с UCI ML Repository
url = 'https://archive.ics.uci.edu/ml/machine-learning-databases/00321/LD2011_2014.txt'
df = pd.read_csv(url, sep=';', decimal=',')
```

**Применение:** Прогнозирование спроса, работа с множественной сезонностью.

---

## 4. Weather Data (Погодные данные)

**Описание:** Метеорологические данные (температура, осадки, давление и т.д.).

**Характеристики:**
- **Период:** Различный (от нескольких лет до десятилетий)
- **Частота:** Дневная/часовая
- **Тип:** Многомерный
- **Переменные:** Температура, влажность, давление, скорость ветра, осадки

**Загрузка:**
```python
# Пример с NOAA или другими источниками
# Можно использовать библиотеку meteostat
from meteostat import Daily, Point
location = Point(55.75, 37.62)  # Москва
data = Daily(location, start, end)
df = data.fetch()
```

**Применение:** Прогнозирование погоды, работа с экстремальными значениями.

## 5. Stock Prices (Цены акций)

**Описание:** Исторические данные о ценах акций компаний.

**Характеристики:**
- **Период:** Различный
- **Частота:** Дневная (торговые дни)
- **Тип:** Многомерный (Open, High, Low, Close, Volume)
- **Особенности:** Волатильность, тренды, отсутствие явной сезонности

**Загрузка:**
```python
import yfinance as yf
df = yf.download('AAPL', start='2020-01-01', end='2023-12-31')
```

**Применение:** Финансовое прогнозирование, работа с волатильностью, технический анализ.

## 6. ETT Dataset (Electricity Transformer Temperature)

**Описание:** Температура электрических трансформаторов и нагрузка. Часто используется в исследованиях по прогнозированию временных рядов.

**Характеристики:**
- **Период:** 2 года
- **Частота:** Часовая (ETTh) или 15-минутная (ETTm)
- **Тип:** Многомерный (7 переменных)
- **Переменные:** Oil temperature (OT), High/Medium/Low UsefulLoad (HUFL, MUFL, LUFL), High/Medium/Low UseOil (HULL, MULL, LULL)

**Загрузка:**
```python
# Доступен на GitHub
url = 'https://raw.githubusercontent.com/zhouhaoyi/ETDataset/main/ETT-small/ETTh1.csv'
df = pd.read_csv(url)
```

**Применение:** Современные методы прогнозирования, бенчмаркинг моделей.

## Рекомендации по выбору датасета

### Для начинающих:
1. **Airline Passengers** - простой, с явными паттернами
2. **Bike Sharing** - интересный, с экзогенными переменными
3. **Weather Data** - понятный, практически применимый

### Для среднего уровня:
1. **Air Quality UCI** - многомерный, требует обработки пропусков
2. **Electricity Consumption** - множественная сезонность
3. **Retail Sales** - бизнес-ориентированный

### Для продвинутых:
1. **ETT Dataset** - бенчмарк для современных моделей
2. **Stock Prices** - сложная динамика, волатильность
3. **COVID-19** - нестационарность, экспоненциальный рост

## Критерии выбора датасета

При выборе датасета для лабораторной работы учитывайте:

1. **Размер:** Достаточно данных для обучения и тестирования (минимум 100-200 наблюдений)
2. **Сезонность:** Наличие сезонных паттернов для практики декомпозиции
3. **Тренд:** Явный или скрытый тренд для практики преобразований
4. **Экзогенные переменные:** Дополнительные факторы для feature engineering
5. **Качество данных:** Наличие пропусков, выбросов для практики обработки
6. **Практическая значимость:** Реальная бизнес-задача для мотивации
