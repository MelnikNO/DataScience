## 📊 Датасет: Титаник

### Описание данных

Датасет содержит информацию о 891 пассажире Титаника, включая:

| Признак | Тип | Описание |
|---------|-----|----------|
| **PassengerId** | int64 | Уникальный идентификатор пассажира |
| **Survived** | int64 | Статус выживания (0 - нет, 1 - да) |
| **Pclass** | int64 | Класс каюты (1, 2, 3) |
| **Name** | object | Полное имя пассажира |
| **Sex** | object | Пол (male/female) |
| **Age** | float64 | Возраст в годах |
| **SibSp** | int64 | Количество братьев/сестер/супругов на борту |
| **Parch** | int64 | Количество родителей/детей на борту |
| **Ticket** | object | Номер билета |
| **Fare** | float64 | Стоимость билета |
| **Cabin** | object | Номер каюты |
| **Embarked** | object | Порт посадки (C, Q, S) |

### Статистика данных

- **Всего записей:** 891
- **Всего признаков:** 12
- **Пропущенные значения:**
  - Age: 177 (19.9%)
  - Cabin: 687 (77.1%)
  - Embarked: 2 (0.2%)

### 2. Работа с данными и фильтрация

```python
# Выборка данных
```
