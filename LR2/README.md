# Анализ данных медицинского страхования с использованием машинного обучения

## Обзор проекта

Этот проект представляет собой полный анализ данных о медицинском страховании с целью построения модели для прогнозирования стоимости страховки (`charges`) на основе характеристик клиента. Работа выполнена в строгом соответствии с методологией **CRISP-DM** (Cross-Industry Standard Process for Data Mining).

**Цель проекта:**
Построить модель машинного обучения, которая будет точно прогнозировать стоимость медицинской страховки, чтобы помочь страховой компании в ценообразовании и управлении рисками.

**Критерии успеха:**
*   Коэффициент детерминации **R² > 0.75** (модель объясняет более 75% вариации стоимости страховки).
*   Среднеквадратичная ошибка **RMSE < 5000** (ошибка предсказания в пределах $5,000).

**Ключевые технологии:**
*   **Язык:** Python
*   **Основные библиотеки:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SciPy, Statsmodels.
*   **Методология:** CRISP-DM.

---

## 1. Понимание бизнес-задачи и данных

### Данные
Для анализа использовался датасет `insurance.csv`, содержащий 1338 записей о клиентах страховой компании. Данные включают 7 признаков:

| Признак | Тип | Описание |
| :--- | :--- | :--- |
| `age` | Числовой | Возраст клиента (от 18 до 64 лет) |
| `sex` | Категориальный | Пол клиента (`male` / `female`) |
| `bmi` | Числовой | Индекс массы тела |
| `children` | Числовой | Количество детей |
| `smoker` | Категориальный | Статус курильщика (`yes` / `no`) |
| `region` | Категориальный | Регион проживания |
| `charges` | Числовой | **Целевая переменная — стоимость страховки** |

### Ключевые выводы из разведочного анализа (EDA)

1.  **Целевая переменная (`charges`):** Распределение имеет правостороннюю асимметрию и не является нормальным. Это было подтверждено тестом Шапиро-Уилка (p-value < 0.05).

2.  **Корреляционный анализ:**
    *   Показал, что `smoker` и `age` имеют наибольшее влияние на стоимость страховки.
    *   `bmi` и `children` имеют слабую положительную корреляцию.
    *   Признаки `sex` и `region` оказались статистически незначимыми для предсказания `charges`.

3.  **Статистические тесты:**
    *   Подтвердили, что разница в стоимости страховки между курящими и некурящими клиентами является статистически значимой. Курильщики платят в **~3.8 раза больше**.

## 2. Подготовка данных

Для подготовки данных к моделированию были выполнены следующие шаги:

1.  **Кодирование категориальных признаков:**
    *   `sex`: `male` → 1, `female` → 0.
    *   `smoker`: `yes` → 1, `no` → 0.
    *   `region`: One-Hot Encoding с удалением первого столбца (базовый уровень — `northeast`).

2.  **Создание новых признаков (Feature Engineering):**
    *   `age_group`: Возрастная группа (молодые, средний возраст, старшие).
    *   `bmi_category`: Категория BMI (норма, избыток, ожирение).
    *   `smoker_bmi`: Взаимодействие статуса курения и BMI (важнейший признак для моделей).
    *   `age_smoker`, `age_bmi`: Другие признаки взаимодействия.
    *   `age_squared`, `bmi_squared`: Полиномиальные признаки.

3.  **Проверка мультиколлинеарности (VIF):**
    *   Обнаружена высокая мультиколлинеарность, особенно для полиномиальных признаков. Это не стало критичной проблемой, так как регуляризованные модели и деревья решений устойчивы к этому.

4.  **Разделение на обучающую и тестовую выборки:**
    *   Выборка была разделена в соотношении **80/20**. Обучающая выборка — 1070 записей, тестовая — 268 записей.

5.  **Масштабирование признаков:**
    *   Для линейных и регуляризованных моделей (Ridge, Lasso, ElasticNet) данные были стандартизированы с помощью `StandardScaler`. Модель Random Forest не требует масштабирования.

---

## 3. Моделирование и результаты

Было построено и обучено пять различных моделей. Для линейных и регуляризованных моделей был проведен подбор гиперпараметров с помощью `GridSearchCV` и 5-кратной кросс-валидации.

### Сравнительная таблица моделей

| Модель | Лучшие гиперпараметры | R² (Test) | RMSE (Test) | Примечание |
| :--- | :--- | :--- | :--- | :--- |
| **Linear Regression** | - | 0.8682 | $4,523 | Базовая модель для сравнения. Хорошо генерализует. |
| **Ridge Regression** | `alpha: 1` | 0.8686 | $4,516 | L2-регуляризация, результат аналогичен линейной регрессии. |
| **Lasso Regression** | `alpha: 10` | 0.8687 | $4,516 | L1-регуляризация. Выполнил отбор признаков, обнулив 3 из них. |
| **ElasticNet** | `alpha: 0.001`, `l1_ratio: 0.1` | 0.8686 | $4,516 | Комбинация L1 и L2 регуляризации. Результат аналогичен. |
| **Random Forest** | `max_depth: 10`, `min_samples_leaf: 4`, `n_estimators: 100` | **0.8732** | **$4,438** | **Лучший результат**. Ансамбль деревьев решений. |

### Анализ важности признаков (Random Forest)

Наибольший вклад в предсказание вносят следующие признаки:
1.  **`smoker_bmi` (81.3%)** — взаимодействие статуса курения и индекса массы тела.
2.  **`age` (4.8%)** — возраст клиента.
3.  **`age_squared` (4.7%)** — квадрат возраста.
4.  **`age_bmi` (4.2%)** — взаимодействие возраста и индекса массы тела.

## 4. Выводы и рекомендации

### Итоги
*   **Цели достигнуты.** Лучшая модель **Random Forest** успешно достигла целевых показателей:
    *   **R² = 0.873**, что означает, что модель объясняет более **87%** вариации стоимости страховки.
    *   **RMSE ≈ $4,438**, что укладывается в целевой порог в $5,000.

*   **Главный фактор:** Самым сильным предиктором является не сам факт курения, а его взаимодействие с индексом массы тела (`smoker_bmi`). Это говорит о том, что курильщики с избыточным весом или ожирением имеют значительно более высокие медицинские расходы.

*   **Второстепенные факторы:** Возраст и его производные (`age_squared`, `age_bmi`) также играют значительную роль, что логично, так как с возрастом увеличиваются риски для здоровья.

*   **Менее значимые факторы:** Признаки `sex`, `region`, а также `children` и `bmi` в отдельности оказали относительно небольшое влияние на итоговую стоимость.

### Рекомендации для бизнеса
1.  **Целевое ценообразование:** Компания может использовать модель для таргетирования и дифференциации цен. Особое внимание следует уделять клиентам, которые одновременно являются курильщиками и имеют высокий индекс массы тела (BMI).
2.  **Управление рисками:** Модель позволяет не только прогнозировать стоимость, но и количественно оценивать вклад каждого фактора в итоговую цену. Это помогает лучше понимать и управлять профилем риска портфеля.
3.  **Маркетинговые кампании:** Понимание портрета "дорогого" клиента может быть использовано для разработки превентивных программ здоровья (например, программы по отказу от курения или снижению веса), что в долгосрочной перспективе позволит снизить общий уровень страховых выплат.
